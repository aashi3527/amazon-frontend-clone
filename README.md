# Amazon Frontend Clone

A front-end clone of the Amazon homepage built with HTML and CSS, replicating the core UI layout and structure.

## 🖥️ Live Preview

https://aashi3527.github.io/amazon-frontend-clone/

## ✨ Features

- **Navbar** — Amazon-style navigation bar with logo, search bar, and icons
- **Footer** — Multi-column footer replicating Amazon's layout and link structure
- Fully built with plain HTML and CSS — no frameworks or libraries

## 🚀 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/aashi3527/amazon-frontend-clone.git
   ```
2. Open `index.html` in your browser
